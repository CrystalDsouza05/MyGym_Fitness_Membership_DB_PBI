# 🏋️ MyGym Fitness Analytics Dashboard

## [Dashboard Link](https://app.powerbi.com/view?r=eyJrIjoiZTNmODQ3MjYtOWMwYS00NWNiLTgwNWQtYjg4MTMzZTg3OTcxIiwidCI6IjQ2NTRiNmYxLTBlNDctNDU3OS1hOGExLTAyZmU5ZDk0M2M3YiIsImMiOjl9)

---

### This Power BI dashboard explores gym membership, revenue, and engagement trends across multiple locations. The analysis aims to support decision-making for improving member retention, optimizing subscription offers, and tracking visit behavior patterns.

---

## 📌 Project Purpose

To analyze and visualize fitness center data across California - helping stakeholders understand user preferences, revenue performance, and engagement to improve operational efficiency and business outcomes.

---

## 🧩 Key Features

- 📊 **KPI Cards** with dynamic arrow-based conditional formatting for MOM and YOY changes  
- 🧠 **Advanced DAX logic** for Total Members, Revenue, and Churn Risk  
- 📍 **Geo-mapped revenue by location**  
- 📆 **Visit frequency patterns** and time-based trends  
- 🕐 **Check-in and workout duration analysis**  
- 📈 **Cumulative revenue growth line chart**  
- 🔄 **Interactive slicers** to filter each page. 

---

## 🧮 DAX Measures Used

### ✅ KPI and Arrow Metrics
- `Total Members`  
- `Total Revenue`  
- `Churn Risk (%)`  
- `Avg Revenue per Member`  
- `Avg Duration (mins)`  
- `Formatted Check-in Time`  
- `Rolling Average Check-ins`  
- `ARROW MOM Revenue %`  
- `ARROW MOM Revenue Change`  
- `ARROW YOY Revenue %`  
- `ARROW YOY Revenue Change`  

### 📊 Engagement Metrics
- `PT Users`  
- `Group Class Users`  
- `Multi-location Members`  
- `Avg Visits per Week`  

### 🔁 Trend Metrics
- `MOM Members Change`  
- `MOM Members %`  
- `YOY Members Change`  
- `YOY Members %`  
- `Same Day Last Month Revenue`  
- `Cumulative Revenue`

---

## 🛠 Tools & Technologies

- **Power BI** for dashboard development  
- **DAX** for custom measures and time intelligence  
- **Data modeling** using a star schema with related dimension tables  
- **Conditional formatting** using UNICHAR arrows for trend indicators  
- **Custom tooltip pages** and visual layout optimization  
- **Dark Theme (Black)** for sleek design presentation
- **Figma and Power Point** for Dashboard BG design and Information Window
